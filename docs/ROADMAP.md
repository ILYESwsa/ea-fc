# BloxCraft 5-Phase Rebuild

## Phase 1
- Monorepo setup with TypeScript client/server/shared packages.

## Phase 2
- Node WebSocket authoritative multiplayer server.
- Chunk generation with biomes/caves and LOD awareness.

## Phase 3
- Three.js client with streaming chunks, instanced rendering, and HUD.

## Phase 4
- Survival/creative sync, inventory, crafting, furnace systems, mobs, persistence hooks.

## Phase 5
- Tests, scripts, docs, and production hardening baseline.
